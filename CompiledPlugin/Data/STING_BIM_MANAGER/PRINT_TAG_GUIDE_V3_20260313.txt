## SHEET 1: TAG FORMAT — 8-Segment ISO 19650 Tag Structure
## ═══════════════════════════════════════════════════════════════════════════════
SECTION,TAG_FORMAT
SEGMENT,POSITION,PARAMETER,EXAMPLE,MAX_LENGTH,VALIDATION,DESCRIPTION,AUTO_POPULATED,MANUAL_INPUT_GUIDE
DISC,1,ASS_DISCIPLINE_COD_TXT,M,3,Must be in ValidDiscCodes set,Discipline code — identifies engineering discipline,YES — from category mapping via TagConfig.DiscMap,"Select from: M (Mechanical), E (Electrical), P (Plumbing), A (Architectural), S (Structural), FP (Fire Protection), LV (Low Voltage), G (Generic). System-aware correction applies for pipes."
LOC,2,ASS_LOC_TXT,BLD1,4,Must be in TagConfig.LocCodes,Location/building code — identifies physical building or area,YES — from room name/number or Project Info via SpatialAutoDetect,"Enter building code: BLD1, BLD2, BLD3 for buildings. EXT for external. XX if unknown. Can also set via Project Information > ASS_LOC_TXT parameter."
ZONE,3,ASS_ZONE_TXT,Z01,3,Must be in TagConfig.ZoneCodes,Zone code — identifies zone within a building,YES — from room Department field or name patterns,"Enter zone: Z01-Z04 for specific zones. ZZ for unzoned areas. XX if unknown. Set room Department field to auto-detect (e.g. 'Zone 1' → Z01)."
LVL,4,ASS_LVL_COD_TXT,L02,4,Must match level code pattern,Level code — identifies floor level,"YES — from Revit level name via GetLevelCode (L01, GF, B1, RF)","Derived from level name. GF=Ground, B1-B9=Basement, L01-L99=Upper levels, RF=Roof, MZ=Mezzanine, PL=Plant, PH=Penthouse, AT=Attic, TR=Terrace, POD=Podium."
SYS,5,ASS_SYSTEM_TYPE_TXT,HVAC,4,Must be in ValidSysCodes set,System type — CIBSE/Uniclass system classification,YES — from MEP system connector traversal or category mapping,"Select from valid CIBSE/Uniclass codes. See SYSTEM CODES section below. Auto-detected from connected MEP systems (6-layer detection)."
FUNC,6,ASS_FUNC_TXT,SUP,4,Must be in ValidFuncCodes set,Function code — sub-system function classification,YES — from smart sub-function detection based on SYS,"Auto-derived from SYS code. HVAC: SUP/RTN/EXH/FRA. HWS: HTG/DHW. Electrical: PWR/LTG. Manual override available via SetFunc token command."
PROD,7,ASS_PRODCT_COD_TXT,AHU,4,2-4 alphanumeric characters,Product code — identifies equipment/element type,YES — from family name matching (35+ specific codes),"Usually auto-detected from Revit family name. E.g. 'Air Handling Unit' → AHU, 'Distribution Board' → DB. Override manually if family name doesn't match standard codes."
SEQ,8,ASS_SEQ_NUM_TXT,0003,4,Must be positive integer (zero-padded),Sequence number — unique identifier within group,YES — auto-assigned by AssignNumbers or BuildAndWriteTag,"Auto-incremented. Unique within DISC-SYS-LVL group. 4-digit zero-padded (0001-9999). Collision detection ensures uniqueness."

## COMPLETE TAG EXAMPLE: M-BLD1-Z01-L02-HVAC-SUP-AHU-0003
## Reads as: Mechanical | Building 1 | Zone 01 | Level 02 | HVAC System | Supply | Air Handling Unit | Sequence 0003
