# MEP Plan Annotation Symbol Families

This directory holds manually-authored `.rfa` Detail Item families used by
`MepSymbolEngine.PlaceSymbols()` for general MEP plan, section, and RCP views.

**Status: empty — families not yet authored.**

## Where families come from

Two sources feed this directory:

### 1. SymbolLibraryCreator (automated)
Run **TAGS tab → Symbols section → Create All Symbols** in STING.
This generates `.rfa` files from `Data/Symbols/STING_MEP_SYMBOLS.json` etc.
using the Revit API and saves them to:

```
<project>/_BIM_COORD/Families/Symbols/
```

`MepSymbolEngine` searches BOTH that path AND this directory, so families
generated by `Create All Symbols` are found automatically. Copy them here
to make them available across all projects without re-generating.

### 2. Manual authoring
Families listed in `Data/MEP/STING_MEP_SYMBOLS_INDEX.csv` that are not
covered by the JSON definitions (e.g. complex equipment outlines, manufacturer-
specific symbols) should be manually authored from `Metric Detail Item.rft`
following the contract in `Families/ISO6412/STING_ISO_SYMBOL_TEMPLATE.params.txt`.

## Naming convention

Match the `family_filename` column in `STING_MEP_SYMBOLS_INDEX.csv` exactly:
```
STING_SYM_<DISC>_<CODE>.rfa
  e.g. STING_SYM_HVAC_AHU_PLAN.rfa
       STING_SYM_ELEC_PANEL_PLAN.rfa
       STING_SYM_PIPE_PUMP.rfa
```
