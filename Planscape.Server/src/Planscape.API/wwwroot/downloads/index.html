Release notes:

What's Included

👁️
Automatic Folder Watching
Monitors your project directories for new and changed files, syncing metadata to Planscape in the background without interrupting your workflow.
📁
ISO 19650 Folder Structure
Generates and enforces the ISO 19650-compliant CDE folder hierarchy for any project, keeping your naming conventions consistent across teams.
🔗
StingBridge IFC Processing
Reads IFC files exported from Revit or ArchiCAD, extracts asset metadata, and pushes structured BIM data directly to the Planscape API.
🔄
Works with Revit & ArchiCAD
No plugin install required inside the host application — Desktop runs independently and processes IFC exports from any authoring tool.
📶
Offline Queue
Changes made while offline are queued locally and flushed automatically when connectivity is restored — no lost data on unreliable site networks.
🔒
Secure by Default
All traffic uses TLS. Credentials are stored in the OS keychain (Windows Credential Manager / macOS Keychain / libsecret). No plain-text secrets on disk.

System Requirements

Windows Windows 10 (build 19041) or later — 64-bit
macOS macOS 12 Monterey or later — Apple Silicon & Intel
Linux Ubuntu 20.04+ · Fedora 36+ · AppImage (no install needed)
RAM 4 GB minimum · 8 GB recommended for large IFC files
Disk 200 MB for the app · additional space for local sync cache
Network Internet connection to your Planscape server URL
Runtime No Node.js or .NET runtime required — fully self-contained

Getting Started

1
Download the installer for your platform
Choose Windows, macOS, or Linux above. The download starts immediately — no account required to download.
2
Run the installer
On Windows, run the .exe and follow the setup wizard. On macOS, open the .dmg and drag to Applications. On Linux, make the .AppImage executable and launch it.
3
Enter your Planscape server URL
On first launch you will be prompted for the URL of your Planscape server (e.g. https://planscape.yourcompany.com). Contact your BIM manager if you do not have this.
4
Sign in with your Planscape account
Use the same credentials you use on the Planscape web dashboard or mobile app. Your sync will begin automatically once authenticated.
5
Add a project folder to watch
Select the root folder of your project in the Desktop app. Planscape will begin watching for new files and syncing changes to the platform.
