# Planscape

ISO 19650 BIM coordination for the practices that don't want to pay $40 a seat for Autodesk Construction Cloud.

## What you get

- **Revit plugin** — tag, sync, publish models direct from Revit
- **Mobile + web viewer** — three.js scene with offline-first issue tracking
- **ISO 19650 templates** — RFI, NCR, transmittal, BEP, MIDP all pre-built
- **Local invoicing** — UGX, KES, TZS, RWF, NGN, ZAR via Flutterwave; USD/EUR/GBP via Stripe
- **Audit-grade trail** — Postgres-side hash chain over every write
