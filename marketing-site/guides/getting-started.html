Getting started with Planscape

📖 10-minute read · Beginner · Last updated 2026-05-23

Welcome. This guide walks you through everything you need to set up Planscape for your firm: signing up, configuring your tenant, creating your first project, and inviting your team. By the end you'll have a working project with the Revit plugin installed and a coordinator on the mobile app.

What you'll need

  • An email address — yours, not a generic one. You're the tenant owner.
  • Your firm's legal name, billing address, and currency preference.
  • For the Revit plugin: a Windows machine running Revit 2025, 2026, or 2027.
  • For the mobile app: an iOS 16+ or Android 10+ phone.
  • Approximate GPS coordinates of your project site (we'll show you how to find them).

Step 1 — Sign up

Visit planscape.app/signup. You'll be asked for:

  1. Your firm's name — this becomes your tenant. It can be renamed later.
  2. Your name, role, and email — you're the tenant owner. Your email becomes your login.
  3. Country & currency — pick the currency you'd like to be invoiced in.
  4. Password — minimum 12 characters. We don't enforce silly symbol-count rules, but we do block known-breached passwords.
No credit card required. Your 30-day free trial starts immediately. We'll only ask for a payment method when you're ready to upgrade.

Step 2 — Configure your tenant

After signup you land on the tenant dashboard. Two things to do here:

Upload your logo

Settings → Branding. PNG or SVG, 400×400 minimum. This logo appears on every transmittal, RFI, and report your firm generates.

Set your default ISO 19650 vocabulary

Settings → Standards. Pick your default suitability codes, document type codes, and revision schemes. Most projects override these per-project, but having sensible defaults saves time.

Heads up: Once a project is created with a particular code scheme, changing the tenant default won't retroactively change that project. Each project has its own copy.

Step 3 — Create your first project

Click + New Project on the dashboard. The form has three sections:

Project metadata

  • Project name — human-readable.
  • Project code — short identifier (e.g. KIH-2026). Used in document numbers.
  • Client name — the appointing party.
  • Start & target completion dates — used by the 4D scheduling tools later.

GPS coordinates

This is what powers the mobile GPS site map. Two ways to set it:

  • Auto-detect from address — type the site address, we geocode it.
  • Drop a pin — click "Pick on map" and click the site location on the world map.

You can refine the coordinates later. The default zoom on the mobile map is set so that a 200m radius around your pin is visible.

RIBA stage & ISO 19650 context

  • Current RIBA stage — drives drawing template defaults.
  • BEP profile — pick from corporate templates or "blank".
  • CDE state machine — defaults to WIP → Shared → Published → Archived.

Click Create project. You're now in the project workspace.

Step 4 — Invite your team

Project › Team › + Invite member. For each person:

FieldNotes
EmailTheir work email — they'll get an invite link.
RoleOwner, BIM Manager, Project Lead, Coordinator, Viewer, or Client.
External?Tick for contractors / clients from other firms — they won't count against your user cap.

Roles map to permissions as follows:

  • Owner — full access including billing.
  • BIM Manager — full project access, no billing.
  • Project Lead — manages deliverables, transmittals, RFIs.
  • Coordinator — raises issues, captures site data, can't approve deliverables.
  • Viewer — read-only.
  • Client — read-only on Published documents; can comment on transmittals.

Roles can be changed any time from the same screen.

Step 5 — Install the Revit plugin

Skip if no-one on your team uses Revit.

  1. From your project workspace: Tools › Download Revit plugin.
  2. You'll get StingTools.zip. Unzip it.
  3. Copy the contents (StingTools.addin + StingTools.dll + supporting files) to:
    %APPDATA%\Autodesk\Revit\Addins\2026\
  4. Start Revit. You'll see a new STING Tools tab on the ribbon, and a "Planscape" dockable panel on the right.
  5. Click Sign in in the panel and use the same credentials as the web app.

For the full Revit plugin walkthrough, see Installing the Revit plugin.

Step 6 — Install the mobile app

  1. iOS: App Store
  2. Android: Play Store
  3. Open the app, sign in, choose your project.
  4. Tap Pre-sync to download the project for offline use — recommended before going to site.

For the full mobile walkthrough, see Mobile app onboarding.

What's next

You're set up. Common next steps:
