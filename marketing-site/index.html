Frequently asked questions

Does it work without an internet connection?
Yes — offline-first is a core design goal, not an afterthought. The mobile app queues photos, voice notes, issue pins, and CDE status changes locally and replays them automatically when connectivity is restored. The Revit plugin caches the last-known project state so authors can keep working and sync when back online.
Which versions of Revit are supported?
Revit 2025, 2026, and 2027. The plugin ships as a single compiled .addin — one download covers all three versions. Older versions (2023, 2024) are not supported due to API differences in the Revit 2025 release.
How does billing in local currency work?
At signup you pick your currency. East African currencies (UGX, KES, TZS, RWF) and West African (NGN, ZAR) are processed via Flutterwave — mobile money, bank transfer, and card all supported. USD, EUR, and GBP go through Stripe. Invoices are issued in your chosen currency so you never pay a foreign-exchange spread on a software subscription.
Can I migrate from Autodesk Construction Cloud?
Yes. We provide a BCF 2.1 import for issues and a document importer for CDE-structured file sets. Your Revit models come across as-is — the Planscape plugin reads the same .rvt files. Book a migration call and we'll scope the move for your project count and data volume.
Is the data stored in Africa?
The primary database region is EU (Frankfurt), chosen for latency balance across East and West Africa and for GDPR compliance that satisfies international donor reporting requirements. An East Africa (Nairobi) edge replica is on the roadmap for 2026 Q3. If data residency in-country is a hard requirement, ask about the self-host option.
What does "audit-grade trail" mean in practice?
Every write operation appends a record to a hash chain in Postgres. Each record includes the previous record's SHA-256 hash, so the chain can be independently verified — any tampering breaks the sequence. The same mechanism is used in the Revit plugin's local sidecar file. This is court-admissible in most jurisdictions and satisfies ISO 19650-2 §5.6 audit requirements.
Can we self-host Planscape on our own servers?
Yes — the server stack (ASP.NET Core 8 + PostgreSQL + Redis + MinIO) ships as a Docker Compose configuration. Self-host is included in the Enterprise plan. See the self-hosting guide for requirements and a step-by-step deployment walkthrough.
